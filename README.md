# TA診断アプリ — エゴグラム（交流分析）

交流分析（Transactional Analysis / TA）のエゴグラムを、50問のセルフチェックで診断するWebアプリです。**完全オフラインで動作し、回答・結果は端末内にのみ保存され、サーバーには一切送信されません。**

> ℹ️ **限定公開**：このサイトは検索エンジンには表示されません（`noindex` / `robots.txt`）。URLを知っている人だけがアクセスできます。URLの共有範囲にご注意ください。

## 特長
- 50問に答えると、5つの自我状態 **CP / NP / A / FC / AC** のバランスを算出
- 結果を **25クラス**（ゲーム風タイプ）とレアリティで表示
- エゴグラム棒グラフ・レーダーチャートで可視化
- **結果の共有**（スコアのみをURLに載せる方式。氏名などの個人情報は含みません）
- **カード画像の保存**（PNG）
- **家族モード**：最大5人の結果を端末内に保存し、レーダーを重ねて比較
- 単一HTML + 外部依存なし（Google Fontsのみ）。GitHub Pagesにそのまま置けます

## ⚠️ 免責事項
本アプリは交流分析（TA）理論に基づく**セルフチェックツール**であり、医学的・心理学的・臨床的な診断ではありません。結果は自己理解のための参考情報としてご活用ください。

## 採点仕様（正確性について）
このアプリの最重要要件は**数値計算の正確性**です。採点の土台となる仕様は以下で固定されており、自動テストで保証しています。

### 配点
| 回答 | 点 |
|---|---|
| その通り／いつも | 3 |
| まあまあ／ときどき | 2 |
| たまには | 1 |
| いいえ／めったにない | 0 |

各尺度10問・合計 **0〜30点**。尺度スコアは単純合計です。

### 質問→尺度の対応（原典どおり）
| 尺度 | 質問番号 |
|---|---|
| CP（批判的な親） | 3, 5, 10, 14, 20, 23, 27, 29, 39, 44 |
| NP（保護的な親） | 6, 7, 12, 17, 24, 42, 43, 46, 48, 50 |
| A（成人） | 1, 8, 15, 19, 21, 33, 36, 40, 41, 49 |
| FC（自由な子） | 2, 9, 22, 28, 32, 34, 37, 38, 45, 47 |
| AC（順応の子） | 4, 11, 13, 16, 18, 25, 26, 30, 31, 35 |

### クラス判定
統一閾値（高=18以上／低=12以下）による宣言的ルールと、ルール外スコアのプロトタイプ距離割当の2段構えで、**どんなスコアでも必ず25クラスのいずれか1つに確定**します。全 31⁵ = 28,629,151 通りの網羅テストで、フォールバック到達ゼロ・全クラス到達を検証済みです。

### レアリティ
回答一様モデル（各0〜3が等確率）における各尺度分布 `((1+x+x²+x³)/4)¹⁰` から、クラス別の出現確率を厳密計算し、`js/rarity-table.js` に静的テーブルとして格納しています（`node tools/build-rarity.mjs` で再生成）。

## 開発 / テスト
依存ゼロ。Node.js（18以上推奨）だけで動きます。

```bash
# 全テスト（採点照合・境界・クラス判定の全数網羅・共有URL・レアリティ）
node --test

# クラス判定の網羅テストを粗い刻みで速く回す
EGOGRAM_STEP=3 node --test

# （注）テストは `node --test`（引数なし）で test/ を自動検出します。
# `node --test test/` は一部の Node バージョンで失敗します。

# レアリティテーブルの再生成
node tools/build-rarity.mjs
```

CI（`.github/workflows/ci.yml`）が push / PR ごとに `node --test` を実行します。

### ローカルで開く
`index.html` をブラウザで直接開くだけで動作します（`file://` 可）。または簡易サーバー:
```bash
python -m http.server 8000   # → http://localhost:8000
```
